- Pautas de Texto:

El tamaño de las líneas es correcto.
Los números grandes se expresan de forma correcta.
El texto no tiene paréntesis, corchetes, porcentajes y otros signos ortográficos difíciles de entender.
Evitar el uso de caracteres de orden.
El texto se compone de frases cortas.
Las fechas se escriben de forma completa.
El uso de pronombres es correcto.
No utilizar números romanos.
El texto va dirigido a la persona que lo lee.
No utilizar la forma pasiva.
Las oraciones han de tener sujeto.
El texto se compone de frases simples: sujeto + verbo + complementos.
-----------------------------------------------------------------------------------------------------------------------
- Pautas de Maquetación:

El tipo de letra facilita la lectura del texto.
El tamaño de letra debe ser visiblemente bueno y cómodo para la lectura.
No utilizar texto en cursiva.
El texto incluye palabras en negrita de forma poco usada.
El texto tiene pocas palabras subrayadas.
El texto no tiene adornos, colores ni sombras.
El texto utiliza las mayúsculas según las reglas ortográficas.
El color del texto debe ser negro.
El color del fondo debe ser blanco sólido.
La cantidad de palabras en la diapositiva es correcta.
